========
Glossary
========

.. glossary::

   CN
	  short name for Control No.

   DUE DATE
   	  a deadline set for every application according to CSC Citizen Charter.

   MOD
      MC

   NAFD
   	  short name for Network and Facility Division under Regulation Branch.

   NEW
      MC

   PPP
      short hand for Permit to Purchase/Possess

   REN
      MC

   RSL
      short name for Radio Station License

   SOA
	   short name for Statement of Account. This is used in issuance of payment.   

   TP
      short name for Temporary Permit

   WORK FLOW
      NAFD process work flow.
