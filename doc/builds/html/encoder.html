Encoding¶

Assign by an Engineer to process Licenses or Permits required by applications.

This is a segment in work flow to upload the encoded records found in MS-Excel or MS-Word.

Listed below are guides to what to upload depending on the application type.

For MS-Excel the application types are as follows:
  1. NEW
  2. MOD
  3. REN /MOD
  4. Permit to Possess (Storage) only
For MS-Word the application types are the details of Permits encoded such as:
  1. Permit to Purchase only
  2. Permit to Possess only
  3. Demo Permit only
  4. Temporary Permit only

Once you are done signing the printed documents, click the check box and press submit button.

Uploading files¶

For MS-Word:

  1. Click Upload MS-Word found under Action column. You will be forwarded to Permit Files list page.

  2. Click Add Permit button to add new.

    Note

    Only MS-Word is accepted.

  3. You have an option to click Logbook No to edit existing file.

  4. Click Save button once all information are properly supplied.

    Hint

    If you want to save then create new entry click Save and add another. If you want to save then continue working click Save and continue editing

For MS-Excel:

  1. Click Upload MS-Excel found under Action column. You will be forwarded to Logbook page.

  2. Below the Logbook page a button for uploading PPP or RSL can be found. Click the appropriate button for what is required to accomplish.

  3. A pop-window will appear with a submit button. Click the Choose File button to select your file.

    Note

    Only MS-Excel file with pre define column format will be accepted.

  4. Click Load data button to upload the data on the pop up window.

  5. The uploaded file will be displayed for final checking. Verify encoded data according to data type columns.

    Caution

    MS-Excel Date column is in format of mm/dd/yy. MS-Excel Number column should not contain character or special character such as dash or comma.

  6. If the data Is good click the check box to confirm and press submit button.

    Hint

    Uploaded information will be displayed in Logbook page after successful upload.

Adding Records Manually¶

Application filed can contain many equipment. Upon importing RSL or PPP using the Uploading files instruction fail or with other reason you want to add new record you have an option to manually type the information.

Here’s the instruction for adding new Equipment.

  1. Go to Home page then find the name called Equipment.

Important

Supply the information on the following relevant fields found in the Equipment page. All of these fields are required unless stated optional.

  1. Call-Sign

  2. Status

  3. Make/Model

  4. Serial No. - type one serial number per equipment

  5. Power - the numerical power value of the equipment

  6. Unit - the power unit

  7. Bandwidth - the field for bandwidth together with emission type

  8. Equipment Usage - choose between Main as the primary equipment or Protection as a backup equipment

  9. Frequency Range - all values should be decimal or integer.

  10. Transmit/Recieved - all values should be decimal or integer.

  11. Permits - the issued control No Permits.

  12. Public Telecom Entity - type Company name

  13. Antenna Details - type the antenna type

    Important

    Check the particular of antenna information if it match the Antenna details.

Here’s the instruction for adding new Radio Station License.

  1. Go to Home page then find the name called Latest Radio Station License.

Important

Supply the information on the following relevant fields found in the Radio Station License page. All of these fields are required unless stated optional.

  1. Logbook

  2. Public Telecom Entity

    type Company name

  3. Status

  4. Date Issued

  5. License No.

  6. Form Serial No.

  7. Capacity

  8. Class of Station

  9. License to Operate

    auto-fill up fields

  1. Nature of Service

  2. Point of Service

  3. Sitename

  4. Steet

    auto-fill up fields depending on Sitename

  5. City

    auto-fill up fields depending on Sitename

  6. Province

    auto-fill up fields depending on Sitename

  7. Region

    auto-fill up fields depending on Sitename

  8. Longitude

    auto-fill up fields depending on Sitename

  9. Latitude

    auto-fill up fields depending on Sitename

  10. Remarks

  11. Encoder

    add by searching username or first name

  12. Evaluator

    add by searching username or first name

  13. Signatory

    the Deparment Head name, add by searching username or first name

  14. Related Equiments

    click add another Equipment for RSL then search by serial no or call-sign

  15. Related Official Receipts

    click add another Official Receipt for RSL then search by O.R. number.
