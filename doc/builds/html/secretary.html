Secretary¶

About NAFD Secretary: assign by the Network and Facility Division Chief to carry out clerical task such as receiving and despensing official documents and messengerial task.

1 Checking requirements
2 Issuance of SOA

11 Cashier Stamp
12 Release to Secretariat

Adding Log Entry¶

  1. Adding new log can be found in Home page in the list name called Logbook.

    Note

    Adding entry can only be done during working days.

Important

The following fields listed below are required before creating new entry. All blurred fields can not be fill up.

  1. Assign to field - Select from the list of personnel who is in charge of the application.

  2. Status - Select one of the following statuses if the application has met such requirement.

    CHECKING REQUIREMENTS

    has not been check for completeness by the Engr. This also means the application had not been issued a Statement of Accounts.

    ISSUANCE OF SOA

    had been check for completeness by the Engr. And the client is asking for a computation of payments.

    PAYMENT

    the client is currently paying the application.

    EVALUATION

    when the Engineer already signed the application for acceptance matching with Record Officer control no.

    ENDORSEMENT

    for endorsement to other Offices, Department or Division acknowledge by the NAFD Chief.

    ENCODING

    is processed by assign encoder.

    REVIEW

    is verified for correctness and accuracy of entry.

  3. Public Telecom Entity - type Company name

  4. Permit No - if the application contain a Permit for Purchase/Possess attachment please indicate the Permit control no.

  5. Statement of Account - if the application contains a SOA attachments please indicate the SOA control no.

    Note

    Statement of Account is created by an Engineer. If your SOA no is showing up please notify the undersign Engineer who created the attach SOA to upload the file.

  6. Click Save button once all information are properly supplied.

    Hint

    If you want to save then create new entry click Save and add another. If you want to save then continue working click Save and continue editing

Adding O.R. Number¶

  1. Adding new Official Receipt Number can be found in Home page in the list name called Official Receipt.

Important

Supply the information on the following relevant fields found in the Official Receipt page. All of these fields are required unless stated optional.

  1. Public Telecom Entity - type Company name

  2. Official Receipt - (O.R.) number stamp by Cashier in Statement of Account.

  3. Date Paid - the date of payment.

  4. Amount - the total amount paid in Philippine currency.

  5. Validity from - the starting period coverage.

  6. Validity until - the ending period coverage.

  7. Remarks - (optional) any additional notes not qualified from other fields can be place here.

  8. Click Save button once all information are properly supplied.

    Hint

    If you want to save then create new entry click Save and add another. If you want to save then continue working click Save and continue editing

Adding Holiday¶

All declared holidays or no working days will not be counted when setting due date on accepted application.

  1. Adding new Holiday or declared no work day can be found in Home page in the list name called Holiday/No work.

Important

All fields are required.

  1. Day - set the date for Holiday or declared no work.

  2. Time frame - choices are Half day or Whole day.

  3. Description - Give a brief description for the set date.

  4. Click Save button once all information are properly supplied.

    Hint

    If you want to save then create new entry click Save and add another. If you want to save then continue working click Save and continue editing

Payment¶

The status will show up when the application started from Checking Requirements. This process will be forwarded to you after the Engineer issued a SOA.

This is a segment in work flow to create a record for Cashier Stamp. This can be found in the application. Here’s an instruction for Adding O.R. Number.

Once you are done click the check box and press submit button.

Chief Signature¶

This is a segment in work flow requiring NAFD Chief Approval.

Once it is signed by the NAFD Chief click the check box and press submit button.

Director Signature¶

This is a segment in work flow requiring Regulation Branch Chief Approval.

Once it is signed by the Regulation Branch Chief Chief click the check box and press submit button.

Cashier Stamp¶

This is a segment in work flow requiring Cashier Officer to verify the authenticity of payment and mark a cashier stamp with signature for the process License or Permit.

Once it is signed by the Cashier Officer click the check box and press submit button.

Release to Secretariat¶

This is a segment in work flow where application had completed the NAFD process. The file is currently forwarded to the Record Officer for release of License or Permit.

Once it is recieved by the Record Officer click the check box and press submit button.
